## Contest Killer 

### 关于 Contest Killer / About

<img src="https://s1.ax1x.com/2023/03/20/ppN6Fuq.png" alt="pic1" style="zoom:25%;" />

- 基于 .NET，WPF 框架开发的**信息学竞赛离线测评器**

- CCR - Plus 换皮版（测评器内核使用 CCR - Plus 测评器源码）

- 适合较大屏幕使用
